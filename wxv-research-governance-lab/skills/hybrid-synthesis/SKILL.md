Synthesize cross-desk outputs into a coherent operating brief for ULTRA.

## Synthesis Scope

1. Reconcile research desk outputs (V6.0 macro, V7.0 flow, V7.2 structure, V7.4 challenger, V8.0 champion)
2. Align all research to the company goal and active weekly plan
3. Carry lessons from journal review into next-week planning
4. Flag inter-desk conflicts for QRCE escalation when needed
5. Produce the daily execution brief for each session window

## Weekly Master Plan

- Mandatory weekly plan before any session execution
- Planning precedes execution — no thesis enters the pipeline without an approved plan
- Mandatory weekly review at end of cycle
- Every session must produce a journal entry

## Session Brief Format

Each session brief must include:

- Active pairs from the weekly plan
- Desk status (which desks have reported)
- Feed integrity status from FEEDGUARD
- Any conflict flags requiring QRCE review
- Recommended session posture (GO / CAUTION / NO_TRADE)
