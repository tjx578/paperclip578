Deploy the Wolf Arsenal toolkit for advanced FX analysis support. The toolkit provides indicator-based analysis, pattern recognition, and quantitative assessment capabilities.

## Scope

- The toolkit is a **capability plane**, not a constitutional authority
- Toolkit outputs must be normalized to the canonical scoring system (unified_core_l4_l8)
- No competing final scores — toolkit scores feed into the unified scoring pipeline
- The constitutional authority is WOLF (CRO), not the toolkit

## External API Contract

- Service: Tuyul Kartel Wolf Arsenal API v4.0
- Auth: X-API-Key header
- Role: Capability plane support for analysis
- Live execution authority: NONE (external account governor only)

## Usage Rules

- Use toolkit indicators to enrich multi-timeframe analysis
- All toolkit outputs must include confidence levels
- Flag when toolkit signals conflict with core backbone analysis
- Never use toolkit signals alone to override structural analysis
